帮农忙队伍职责声明

如果您是该订单队长，则您须履行如下职责：

1. 接单时，须与队员沟通，并确保全队车辆及配置能符合订单要求。

2. 在确认最终收割日期的2天内与客户联系，并确保全队可根据客户要求按时到达。若由于客户原因导致收割日期延后，请与客户协商并重新制定日期，若无法达成一致，则帮农忙及机手不承担违约责任。

3. 须确认全队成员已阅读并了解相应责任和权利要求。

4. 开割日24小时前，请确保全队机手已打开软件定位功能，以便帮农忙端可了解队伍到位情况。若由于机手原因，导致无法按时到达并无法开割作业，则帮农忙会根据客户要求帮其重新叫车或将机手押金部分作为赔偿款以补偿客户。

5. 到达后，请主动并积极配合客户的试割作业，并确保作业质量达到已约定质量要求或普遍要求。若客户对于作业质量提出异议，队长需与客户沟通并达成一致。若后续仍无法达到要求，经帮农忙根据双方提交的证明判定此问题存在后，客户有权要求更换车辆。

6. 作业过程中，由于机手设备故障无法开展工作，队长需配合客户要求帮其调车，以按时完成作业。

7. 须确保全队机手作业中全程开启软件及定位功能。

8. 每日作业完成后，需与客户确认当日作业亩数，并签订作业亩数签订表，上传作业亩数至帮农忙APP。若对于亩数双方产生异议，请队长与客户进行沟通协商。

9. 作业全程中，接受客户的监督，并确保作业质量达到已约定质量要求或普遍要求。若客户对于作业质量提出异议，队长需与客户沟通并达成一致。若后续仍无法达到要求，经帮农忙根据双方提交的证明判定此问题存在后，客户有权要求更换车辆。

10. 每日配合客户要求至指定田块（田块须为已签约田块）作业。若由于机手原因导致作业无法开展，经帮农忙核实后，帮农忙根据客户要求帮其重新叫车或将机手押金部分作为赔偿款以补偿客户。若由于天气等不抵抗因素导致作业无法开展，则帮农忙及机手不承担违约责任。

11. 若由于客户端原因导致作业停止，队长须配合帮农忙端采集证据。

同时，您享有如下权利：

1. 若田块异常情况（包含但不限于烂地，倒伏）超过合同约定比例，队长可与农户针对作业标准及价格进行协商。若无法达成一致，经帮农忙核实后，机手及帮农忙端不承担违约责任。

2. 若客户的周转能力低于合同约定而导致作业无法正常展开，队长可与农户进行协商。若无法达成一致，经帮农忙核实后，机手及帮农忙端不承担违约责任。

3. 作业完成后，队长可与帮农忙确认全队最终作业亩数，确认作业款结款流程，并要求帮农忙端按时打款。

4. 队长有权分配每日的作业亩数至各队员账户。

5. 对于产品的定金扣除等情况，队长有权决定各队员的定金返还或扣除金额数。

6. 队长对于部分争议项目，可直接与客户协商并作出最终判断。

7 对于需帮农忙端协助的事宜，队长可直接与客服联系，并进行三方协商。

如果您是该订单的接单队员，您须履行如下职责：

1. 接单时，须确保车辆及配置能符合订单要求。

2. 须根据客户要求按时到达。若由于客户原因导致收割日期延后，请与队长联系确认新的时间，若在新时间点无法到达，则帮农忙及机手不承担违约责任。

3. 须阅读并了解相应责任和权利要求。

4. 开割日24小时前，须打开软件定位功能，以便帮农忙端可了解机手到位情况。若由于机手原因，导致无法按时到达并无法开割作业，则帮农忙会根据客户要求帮其重新叫车或将机手押金部分作为赔偿款以补偿客户。

5. 到达后，请主动并积极配合客户的试割作业，确保作业质量达到已约定质量要求或普遍要求，并根据客户提出的异议加以改善。若后续仍无法达到要求，经帮农忙根据双方提交的证明判定此问题存在后，客户有权要求更换车辆。

6. 作业过程中，若设备出现故障，请及时告知队长。

7. 须确保作业中全程开启软件及定位功能。

8. 每日作业完成后，需将当日作业亩数告知队长。

9. 作业全程中，接受客户的监督，并确保作业质量达到已约定质量要求或普遍要求，并根据客户提出的异议加以改善。若后续仍无法达到要求，经帮农忙根据双方提交的证明判定此问题存在后，客户有权要求更换车辆。

10. 每日配合客户要求至指定田块（田块须为已签约田块）作业。若由于机手原因导致作业无法开展，经帮农忙核实后，帮农忙根据客户要求帮其重新叫车或将机手押金部分作为赔偿款以补偿客户。若由于天气等不抵抗因素导致作业无法开展，则帮农忙及机手不承担违约责任。

11. 若由于客户端原因导致作业停止，须配合队长帮农忙端采集证据。

同时，您享有如下权利：

1. 若田块异常情况（包含但不限于烂地，倒伏）超过合同约定比例，请及时反馈并告知队长。

2. 若客户的周转能力低于合同约定而导致作业无法正常展开，请及时反馈并告知队长。

3. 作业完成后，可要求帮农忙端按时打款。

4. 对于作业过程或后续有异议时，请及时反馈并告知队长。

注意：如果您成功抢单后，未能按照以上权责声明履行义务，则帮农忙有权利中止该订单的合作，并且根据服务农户的协商沟通结果，依据平台规则做出相应处理。
